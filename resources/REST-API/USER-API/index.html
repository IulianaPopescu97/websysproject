AddUserInDatabase

Up 
post /sign-up
Sign-up user (addUser)
Add new user in Firebase.

Return type

array[sign-up]

Example data

Content-Type: application/json
[ {
  "password" : "Ld53v-9k",
  "confirmPassword" : "Ld53v-9k",
  "email" : "iuliana_p92@yahoo.com"
}, {
  "password" : "Ld53v-9k",
  "confirmPassword" : "Ld53v-9k",
  "email" : "iuliana_p92@yahoo.com"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successful sign-up.

400

The email/password has a bad format.

409

User already exist.

ChangePassword

Up 
post /forgot-password
User forgot password (changePassword)
User forget the password and needs to change it.

Return type

array[forgot-password]

Example data

Content-Type: application/json
[ {
  "email" : "iuliana_p92@yahoo.com"
}, {
  "email" : "iuliana_p92@yahoo.com"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Password reset email sent, check your inbox.

404

There is no user record corresponding to this identifier.
